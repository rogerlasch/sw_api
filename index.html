Introdução

Esta API foi desenvolvida como parte de um exercício prático na aula de Serviços Web, abordando os conteúdos discutidos em sala de aula. Ela representa uma API de Pedido Musical, permitindo que os usuários registrem solicitações de músicas para que um radialista as atenda posteriormente. Para fazer um pedido, o usuário deve fornecer informações como o nome da música, o nome do artista, o seu próprio nome e, opcionalmente, um recado.

GET /pedidos

Retorna a lista de pedidos de música.

POST /pedido/create

Cria um novo pedido de música.

Parâmetros: JSON contendo musicName, artist, listener e msg.

PUT /pedido/update

Atualiza um pedido de música existente.

Parâmetros: JSON contendo id, musicName, artist, listener, msg, status e timestamp.

DELETE /pedido/delete/{id}

Exclui um pedido de música pelo ID.

Parâmetros: Id - O ID do pedido a ser excluído.
